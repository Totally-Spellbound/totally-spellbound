# Requirements: Totally Spellbound

**Defined:** 2026-03-22
**Core Value:** A thoughtful, independent e-commerce experience that gently guides customers to the right products — helpful like a conversation in the shop, never pushy.

## v1 Requirements (SLC — Simple, Lovable, Complete)

### Foundation

- [x] **FOUND-01**: Clean skeleton structure for all core pages (home, product, collection, cart, search)
- [x] **FOUND-02**: Shopify theme editor compatibility — sections and blocks configurable via admin
- [ ] **FOUND-03**: Mobile-responsive layout — most traffic will come from social media on phones
- [ ] **FOUND-04**: Performance: pages load fast on mobile (< 3s LCP)

### Design System

- [ ] **DSGN-01**: Dark color palette — deep black base with purple/violet accents and warm gold touches
- [ ] **DSGN-02**: Light/dark theme toggle — persistent preference, accessible from all pages
- [ ] **DSGN-03**: Typography system — serif display headings (mystical feel), clean sans-serif body, generous scale
- [ ] **DSGN-04**: Handmade, non-template visual identity — must not look like a stock Shopify theme
- [ ] **DSGN-05**: Consistent component styling (buttons, cards, inputs) with "Add to Cauldron" branding language

### Homepage

- [ ] **HOME-01**: Hero section with subtle animation/interaction (cursor-reactive glow or parallax, not annoying)
- [ ] **HOME-02**: Featured/new products section with bento-style varied cards (not a uniform grid)
- [ ] **HOME-03**: Shop by star sign entry point — curated, feels personalized
- [ ] **HOME-04**: "Proudly independent" messaging — support a real person, not a corporation
- [ ] **HOME-05**: Physical shop awareness — location, personality, "come visit us"
- [ ] **HOME-06**: Scroll-triggered animations — sections fade/float in as you scroll down
- [ ] **HOME-07**: Clear, prominent search access from homepage

### Search

- [ ] **SRCH-01**: Omni-search bar — prominent, always accessible from header
- [ ] **SRCH-02**: Instant search results as you type — shows products with images, prices, and category
- [ ] **SRCH-03**: "You might also like" suggestions alongside search results — gentle, not pushy
- [ ] **SRCH-04**: Search works for product names, categories, crystal types, and properties

### Product Pages

- [ ] **PROD-01**: Product display with high-quality imagery (Photoroom-cleaned photos)
- [ ] **PROD-02**: "Add to Cauldron" button with satisfying interaction feedback
- [ ] **PROD-03**: Crystal/product properties shown (what it's for, how to use it)
- [ ] **PROD-04**: Complementary product suggestions — "goes well with" (not aggressive upselling)
- [ ] **PROD-05**: Personalised/custom product support — customer can add engraving/customisation request

### Collections

- [ ] **COLL-01**: Star sign / zodiac collections — browsable, curated feel
- [ ] **COLL-02**: Category collections (crystals, incense, gifts, etc.) with clear navigation
- [ ] **COLL-03**: Collection pages match design system — not default Shopify grid
- [ ] **COLL-04**: Filter and sort functionality within collections

### Cart & Checkout

- [ ] **CART-01**: Cart page styled to match theme ("Your Cauldron" branding)
- [ ] **CART-02**: Shipping tiers visible: £2.50 small / £4.50 larger / free over £30
- [ ] **CART-03**: Cart accessible from all pages with item count indicator

### Events

- [ ] **EVNT-01**: Events page for craft fairs, live sessions, and markets
- [ ] **EVNT-02**: Visual timeline or flowing layout — not a plain list
- [ ] **EVNT-03**: Alison can update events via Shopify admin (section-based, no code needed)

## v2 Requirements (Post-SLC)

### AI Oracle

- **ORAC-01**: AI-powered search helper with crystal knowledge (e.g., "I need protection" → suggests obsidian, selenite, tourmaline)
- **ORAC-02**: Conversational product discovery — witchy-themed chat interface
- **ORAC-03**: Knowledge base of crystal properties, uses, and complementary pairings

### Celestial Integration

- **CELS-01**: Moon phase / planetary position display on homepage
- **CELS-02**: Dynamic product recommendations tied to current astrological events
- **CELS-03**: "This month's energy" editorial content block — written by Alison
- **CELS-04**: Seasonal event awareness (Samhain, Yule, Solstices, Full Moons)

### Educational Resources

- **EDUC-01**: Crystal guide pages — properties, uses, care instructions per crystal type
- **EDUC-02**: Star sign guide pages — traits, compatible crystals, seasonal advice
- **EDUC-03**: Blog/journal for Alison's insights and seasonal content

### Enhanced Personalisation

- **PERS-01**: Dedicated showcase page for laser engraving and 3D printing services (not just product listings)
- **PERS-02**: Custom order request form with preview/mock-up capability
- **PERS-03**: Gallery of past custom work

## Out of Scope

| Feature | Reason |
|---------|--------|
| Mobile app | Shopify handles mobile web responsiveness. No budget/need for native app |
| Multi-currency / international shipping | Start UK-only, expand later based on demand |
| Customer accounts / login | Shopify guest checkout is sufficient for v1. Accounts add complexity |
| eBay / Etsy integration | Ruled out due to fees. Revisit in v2 if needed |
| Real-time chat (live customer support) | Alison has max 30 mins/week for tech. Can't staff live chat |
| Subscription boxes | v2+ idea from growth plan, not SLC |
| Video content hosting | Use YouTube/Facebook for lives, link from site |
| Loyalty program | v2 — needs customer accounts first |
