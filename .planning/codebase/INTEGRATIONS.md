# External Integrations

**Analysis Date:** 2026-03-22

## APIs & External Services

**Shopify Platform APIs:**
- Storefront API - Built-in for product browsing, cart management, checkout
- Theme API - Native Shopify Liquid API for theme development
- Admin API - Used via Shopify CLI for theme management and deployment

**Payment Processing:**
- Shopify Payments - Native payment processing (configured in store settings)
- Payment Terms Widget - `shopify-payment-terms` web component for installment options
  - Rendered in `snippets/cart-summary.liquid`
  - Accessible via shadowRoot in `assets/cart-drawer.js`

## Data Storage

**Databases:**
- Shopify Managed Database - All product, order, and customer data stored in Shopify
- Connection: Native Shopify store (`totally-spellbound-2.myshopify.com`)

**File Storage:**
- Shopify CDN - Static asset serving and image optimization
- Image URL filters: `image_url`, responsive image tags
- Asset URL tags: `asset_url` filter for JavaScript and CSS files

**Cart & Session:**
- Shopify Cart - Built-in cart object accessed via Liquid
- Browser LocalStorage - For client-side features like recently viewed products
- Predictive Search Cache - Client-side caching in `assets/predictive-search.js`

## Authentication & Identity

**Auth Provider:**
- Shopify Native - Customer accounts managed by Shopify
- Implementation: Shopify's customer login system integrated via Liquid
- Endpoints: `/account`, `/account/login`, `/account/register` (Shopify-provided)

**Staff/Collaborator Access:**
- Shopify Partners Account - For theme development and store access
- CLI Authentication: `shopify auth logout` / re-login flow documented in `README.md`

## Monitoring & Observability

**Error Tracking:**
- None detected - No external error tracking service configured

**Logs:**
- Shopify Admin Logs - Store activity and theme deployment logs
- Browser Console - Client-side error logging via vanilla JavaScript

**Performance Monitoring:**
- Shopify Theme Performance - Built-in theme performance metrics
- Custom timing: `cartPerformance` object in `assets/performance.js`

## CI/CD & Deployment

**Hosting:**
- Shopify Managed Hosting - Theme deployed to Shopify's infrastructure

**Deployment Tools:**
- Shopify CLI - Primary deployment mechanism
- Commands:
  - `npm run dev` - Deploy as development theme with hot reload
  - `npm run push` - Push to production theme
  - `npm run pull` - Sync changes from store
  - `npm run share` - Generate shareable preview link

**Theme Repository:**
- GitHub: `https://github.com/Totally-Spellbound/totally-spellbound`
- Branch-based deployment via Shopify CLI

## Environment Configuration

**Required Environment:**
- Store handle: `totally-spellbound-2`
- Theme store context configured in npm scripts
- Shopify CLI authentication via browser login flow

**Secrets Location:**
- Shopify Admin Dashboard - API credentials stored in Shopify's managed environment
- `.env` files: Not used (Shopify CLI handles authentication via session tokens)

**Configuration Files:**
- `config/settings_schema.json` - Theme settings schema exposed in Shopify admin
- `config/settings_data.json` - Current setting values (auto-generated by Shopify)

## Webhooks & Callbacks

**Incoming:**
- Theme Editor Events - Shopify Liquid events for theme customization
  - `shopify:block:select` - Block selection in theme editor
  - `shopify:block:deselect` - Block deselection
  - `shopify:section:load` - Section loaded (in `assets/theme-editor.js`)
  - `shopify:section:unload` - Section removed

**Outgoing:**
- None detected - This is a frontend-only theme with no backend webhooks

## Third-Party Scripts

**Analytics & Tracking:**
- None detected - No Google Analytics, Facebook Pixel, or similar configured
- `content_for_header` variable in layout allows Shopify to inject scripts (in `layout/theme.liquid` line 39)

**Social Media:**
- Social icons available: Facebook, Twitter, Instagram (icon helper in `snippets/icon.liquid`)
- Social links configurable in theme settings but no direct social API integration

## Liquid Filters & Global Objects

**Available Shopify Liquid Objects:**
- `shop` - Store information
- `product` - Product data
- `collection` - Collection data
- `cart` - Cart object
- `customer` - Customer account data
- `request` - Request context (locale, page type, origin)
- `localization` - Multi-language and multi-currency support

**Built-in Routes:**
- `routes.cart_add_url` - Add to cart endpoint (JSON)
- `routes.cart_change_url` - Modify cart endpoint
- `routes.cart_update_url` - Update cart endpoint
- `routes.cart_url` - Cart page
- `routes.predictive_search_url` - Search API endpoint
- `routes.search_url` - Search page
